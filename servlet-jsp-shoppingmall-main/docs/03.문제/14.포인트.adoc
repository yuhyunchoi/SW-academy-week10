= 포인트

* 포인트는 주문/결제  구현시 적절한 시점에 구현합니다.
* 주문결제 시 포인트 적립은 독립된 Thread 내에서 동작합니다.
** 반대로 포인트 적립이 실패하더라도 주문결제는 정상 처리 됩니다.
